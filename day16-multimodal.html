A multimodal model has an encoder per modality that projects everything into one shared representation — so it can see, read, hear, and answer across them together. It's the same idea as embeddings, extended beyond text. This demo uses tiny hand-placed points so it's deterministic; real models learn the shared space from millions of paired examples (image+caption, audio+transcript).
