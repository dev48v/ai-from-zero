🔨 Build a RAG pipeline, in code

    
The demo uses tiny hand-authored vectors. A real RAG pipeline embeds text with a model, stores the vectors in a vector database (pgvector, Chroma, Pinecone, FAISS), retrieves the top-k chunks for a question, and feeds them to a text-generation model. ~8 steps with copy buttons.
