🔨 How to reduce hallucination

    
You can't fully eliminate it, but a few habits cut it dramatically.

    
    
      
  1. 1
    Ground it (RAG)
    Give the model the actual documents and instruct it to answer ONLY from them, citing sources — the single biggest fix (see PromptFromZero's RAG days).
    2. 
      
  2. 2
    Give it an out
    Explicitly allow "I don't know." Add: "If the context doesn't contain the answer, say you don't know." Without permission to abstain, it will guess.
    3. 
      
  3. 3
    Lower the temperature
    For factual tasks, set temperature near 0 so it picks the likeliest (usually safer) token instead of gambling on rare ones (tomorrow's topic).
    4. 
      
  4. 4
    Verify the important stuff
    Ask for sources you can check, cross-question, or have a second pass verify claims. Treat confident output as a draft, not gospel — especially names, numbers, citations.
